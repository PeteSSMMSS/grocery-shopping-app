## üìù Best Practices

### 1. Immer Windows zuerst testen
- Mindestens 5 erfolgreiche Tests auf Windows
- Alle Features durchgehen
- Keine Fehler in Logs

### 2. Aussagekr√§ftige Commit Messages



- Windows nur f√ºr Development
